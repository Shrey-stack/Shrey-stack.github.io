Nmap

Nmap is a very powerful utility that can be used to:< br /> Detect the live host on the network (host discovery)< br /> Detect the open ports on the host (port discovery or enumeration)< br /> Detect the software and the version to the respective port (service discovery)< br /> Detect the operating system, hardware address, and the software version< br /> Detect the vulnerability and security holes (Nmap scripts)< br />

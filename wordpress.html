Setting up Wordpress on Amazon EC2

WordPress is open-source software and most popular CMS (content management system) platform used by many developers and entrepreneurs around the world to run their blogs and fully functioning websites. In addition, WordPress is a fully-managed customizable platform with thousands of themes and plugins supported by a large global community.

I assume that your EC2 Ubuntu server has been launched and you're already connected to the remote server.
The first thing you're going to do is install apache to your server. If you manage your website hosting using the WordPress platform, you will need to required installing a web server software.Hence Apche2 tada!

The next step is to install database to store and retrieve every requested data. In this case, we are going to install mariadb-server and mariadb-client within the EC2 Ubuntu instance.
Next, we are going to install mysql_secure_installation.
mysql_secure_installation is a shell script that enables you to improve the security of your MariaDB installation in the following ways:
* You can set a password for root accounts.
* You can remove root accounts that are accessible from outside the local host.
* You can remove anonymous-user accounts.
* You can remove the test database, which by default can be accessed by anonymous users. You will be asked to enter the current password for root, Simply press
ENTER

Then you will be asked to set root's password. Type y
Set a password and make sure you don't forget it. Re-enter your password.
After that you will be asked to remove anonymous users. Press Y
Disallow root login. Press Y
Remove test database and access to it. Press Y
Reload privilege table. Press Y
and Voilà you're done!(for sometime!)
The next step is to install PHP.
After you're done installing the basics which is very similar to LAMP stack, you finally install wordpress.
Oh no! Wait a minute!
A small step to do before installing wordpress and i.e., To install wget and unzip
And finally you go to your web browser and search Download Wordpress or simply press here.
Download Wordpress
So in the blue Download Wordpress 5.5.1,you rightclick it and you copy the link.
Go to the terminal, type sudo wget and paste the link.
After the download is complete, type ls. You will see the zip file there. Then simply type sudo unzip latest+tab key
And ta-da your wordpress is installed! Then you copy the wordpress file to /var/www/html.
First things first, you would want to remove your index.html coz you don't want the apache page to be shown when you open your browser.
Now you've got to change the owner of the group. To check the owner, simply type ls -l.
You can see that all the contents belongs to root.
You would want to give wordpress the file permissions.
So simpley type the command below:
sudo chown www-data:www-data -R /var/www/html/
Now the final thing you would do is to create database and user for it.
So you simply type the command
sudo mysql -u root -p You enter the password you used above while creating the secure installation. I told you to remember it.
Now you're logged in to your database. So the first thing to do here is to create database. Simple type
create database wordpress; (Wordpress is just a name I gave for my database.)
Then you create a user by
create user"wpadmin"@"%" identified by "wpadminpass"; (Here, my username is wpadmin and password is wpadminpass) After that you grant privileges to your user by
grant all privileges on wordpress.* to "wpadmin"@"%";
You're pretty much done! Type exit!
If you think you're finally done, go to EC2 instance and get your public address and view it. If it works YAHOO!!1but if it doesn't then no worries, just a simple thing to do!!!
Go to your EC2 security group, edit your inbound rules and simply add HTTP and HTTPS. In the source, you would simply press anywhere. SAVE IT.
Now go to your browser type your public address and this is what you'll be seeing now. Simple press Let's Go
Type your database name, databaseadmin and the password.
Make no changes to the Database Host and the Table Preflix.
Press Submit
Press Run the installation!
You will be asked for your Site-Title, Username, Password and Email. The username and passwords are not the one you used earlier for your database.
Now, you'll be asked to login
You will now be directed to the Wordpress Dashboard.
If you see above the dashboard slightly towards the left side, you'll see a home icon. Click the icon and press the Visit Site
And Voilà! You're done!!! Congratulations for setting wordpress on your EC2 instance. You completely deserve a pat on the back even if you have to do it Yourself!
